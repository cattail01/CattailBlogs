# cattail blogs 个人博客搭建

## 项目引言

### 项目目标

* 本项目的目标是完成自己的博客开发，顺便保持对编程的手感

### 项目周期

* 本项目的周期会很长，而且更新内容很随缘

### 其他

#### 技术

* 项目采用的技术均为接近最新版

#### 代码

* 如有雷同，均为借鉴，但是肯定会对原有内容做出改动
* 代码风格为从上到下，越靠上为越容易改动的属性值，靠下的为该类的重要功能方法，下面的方法会使用上面的代码，按照开发顺序进行阅读即可
* 注释冗长的原因，一方面是查的资料均能在注释体现，一方面是复制一些内容便于~~抄袭~~借鉴，还有一方面是不会轻易delete已经写好的代码，仅用注释删除可能存在多余的方法，但是遵循了尽量使用方法级注释而不是方法内注释的原则，抽象整个算法或者复杂功能，让主要功能的方法内部的代码更加简洁舒适。
* 还有什么问题就是：别问，问就是代码风格

#### 项目

* 为了发挥java和python不同的优势，特意写了多个项目

* 后期甚至还可能有C#的项目

* 为了简化运行，写了能够一键全部运行的start项目，语言是python，能够自动识别所需环境和库，并给出安装和修改提示【未完成】

  > start项目我觉得是这整一套项目中的点睛之笔
